# 3D_prints_for_private_use
This repository contains 3D printing files created by me for your private use.
When printing the files make sure to check the quality of your print:
- check for artifacts
- check for strings
- check if the selected printing material is suitable for the application (contact with food, temperature resistance)
If needed postprocess the print.

The files cannot be changed or used for commercial use (see section license and license.md file)
